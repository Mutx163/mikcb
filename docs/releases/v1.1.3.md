# 轻屿课表 1.1.3

本次更新：

- 重要说明：应用内已明确标注 HyperOS 3.0.300 起支持超级岛 / 焦点通知，减少系统版本误解
- 优化提醒设置文案，明确“上课前提醒”和“课中 / 临近下课提醒”互不替代，避免误关
- 首页时间列新增三档显示模式：不显示 / 仅显示上课时间 / 显示上下课时间
- 默认将首页时间列改为显示上下课时间，查看作息更直观
- 支持隐藏周六周日，开启后周一到周五会自动拓宽并铺满首页
- 新增应用内震动总开关，可统一关闭页码切换等轻微震动反馈
- 课程颜色支持调色盘自定义，不再局限于预设颜色
- 优化首次使用引导和关于页文案，补充系统支持范围与使用说明

这是对 1.1.2 的体验与说明修复更新，建议升级。
