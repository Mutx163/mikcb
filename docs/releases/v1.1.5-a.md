# 轻屿课表 1.1.5-a

友盟更新：

- 接入友盟移动统计、应用性能监控与高级运营分析依赖
- 首次使用新增友盟隐私同意引导，在引导页完成阅读与勾选后再初始化
- 测试版新增友盟崩溃测试与异常卡顿测试入口，便于验证上报链路

功能更新：

- 修复时间模板可配置跨 0 点节次的问题，避免异常课表时间进入数据
- 调整测试通知逻辑，现在仅发送上课提醒测试
- 修复超级岛测试通知触发时机，测试时更容易稳定看到岛展示
- 优化编辑课程保存后返回首页时课表被瞬间压缩的问题
- 应用内下载更新时，下载进度改为固定显示在页面底部，状态更容易看到
- Android 端暂时禁用 Impeller，用于规避部分设备上的 Vulkan 渲染崩溃
