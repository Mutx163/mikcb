# v1.1.9.6

- 本版本为预发布版本，主要用于验证桌面小组件自定义、深色模式，以及首页更新入口的跳转修复
- 桌面小组件新增卡片高度微调和圆角调节，默认值改为更稳的中间档位
- 修复 `2×2` 和 `4×4` 小组件在“今日课程已结束”状态下重复显示结束文案的问题
- 修复 `2×2` 课程列表在课程已结束后仍显示“还有 X 节”的问题
- 首页“软件有更新”现在会直接跳到版本更新三级页面，不再先落到关于页
- 应用新增深色模式，支持跟随系统、浅色模式和深色模式切换
- 设置首页入口卡片的点击高亮动画收敛，深色模式返回时不再出现明显白色按压残影
- GitHub Pages 项目主页已接入 `docs/` 目录发布，Android 下载流程和仓库展示一并整理
