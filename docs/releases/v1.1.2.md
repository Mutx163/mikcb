# 轻屿课表 1.1.2

本次更新：

- 重要修复：修正 WakeUp 日历导入时的单双周识别问题，原本被拆成多段的双周课程现在会正确合并为单双周课程
- 重要提醒：如果你之前已经导入过受影响的课表，请先清空当前课表后重新导入，旧数据不会自动修正
- 修复部分周六课程被误扩到下一周的问题，避免第 7 周出现本不该显示的双周课程
- 新增多课表管理，支持创建、复制、切换、重命名、删除和清空当前课表课程
- 新增全局时间模板库，支持给学校作息命名、切换模板，并通过快捷生成快速配置整套节次时间
- 新增课程冲突识别与首页冲突小胶囊，课程总览会按课程集合显示冲突信息
- 优化课程总览中的冲突显示，冲突信息直接显示在课程集合标题上
- 新增编辑课程页右上角删除按钮，删除课程更直接
- 新增首页更新提醒入口，检测到新版本时可从右上角菜单直接进入关于页查看更新
- 新增全部数据备份，支持导出和恢复所有课表、当前选中课表以及时间模板
- 默认开启实时提醒的前缀隐藏，并优化时间模板快捷生成的输入记忆与大课间规则配置

这是对 1.1.1 的稳定性与导入兼容性修复更新，建议升级。
