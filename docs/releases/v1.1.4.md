# 轻屿课表 1.1.4

本次更新：

- 新增小米超级岛左侧文字图标实验功能，可把课程名、教室或课程名+教室显示到左侧
- 支持左侧文字仅文字 / 图标+文字两种样式，并可自定义文字大小
- 优化左侧文字图标渲染：更紧凑的留白、更高的清晰度、更合理的图标比例
- 调整左侧文字宽度策略，优先自动缩字，减少过早出现省略号
- 新增上课中紧凑提醒时间显示模式，可二选一显示最近时间或总时间
- 修复修改超级岛左侧文字相关设置后桌面岛状态不立即刷新的问题，保存后会强制重建当前 Live Update
- 优化时间模板入口、周次选择器和更新页排版
- 修复部分 `.ics` 导入问题，包括双周课识别与教室解析

说明：

- 小米超级岛左侧文字图标属于实验性能力，仅在小米系设备样式上生效
- 如果你之前已经遇到过导入单双周异常或教室解析不完整，建议清空受影响课表后重新导入
