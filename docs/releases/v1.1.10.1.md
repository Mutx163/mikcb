# v1.1.10.1

- 更换应用包名为 `com.mutx163.qingyu`，并补齐 Android `namespace`、Kotlin 包路径与 MethodChannel 前缀重命名
- 新版首启支持识别旧版应用，提供包名迁移引导页
- 迁移页补充旧版备份步骤说明，明确使用 `导出全部数据`
- 迁移页增加下载目录与微信中转保存说明，固定底部按钮，其余内容改为滚动
- 新版从迁移页或新用户欢迎页恢复备份时，首次导入不再弹出“覆盖/新课表”选择
- 修复首次看过迁移页后直接关闭应用，再打开被误判为新用户的问题
- 小米自启动入口恢复优先打开真正的自启动管理页
- 辅助功能入口改为优先尝试直达当前应用服务详情页，失败再回退总辅助功能设置
- Debug 包保持独立测试包：`com.mutx163.qingyu.debug`
