# v1.1.9.7

- 本版本为预发布版本，主要用于验证超级岛倒计时格式个性化、课中普通通知恢复，以及顶部区域卡住问题的修复
- 超级岛倒计时支持更多格式：`智能（中文）`、`智能（英文）`、`5分钟19秒`、`5min19s`、`5min/19s`、纯分钟、纯秒等
- 上课前提醒显示和课中/下课提醒显示可分别设置各自的倒计时格式
- 纯分钟样式改为按文案真正变化时刷新，带秒样式按秒刷新，不再固定一秒一刷
- 修复关闭超级岛后，课中普通通知会一并消失的问题
- 修复非 promoted 普通通知仍携带小米焦点参数，导致顶部状态栏超级岛区域可能卡住的问题
- 移除“上课后先占用顶部状态栏区域”的中间过渡阶段，减少 HyperOS 下状态栏交互异常
