# v1.1.8.2

## 功能更新

- 超级岛设置、外观、布局、桌面小组件改为改动即保存，无需再手动点击保存。
- 时间模板改为统一弹窗链路，编辑页支持自动保存；保存失败会自动回滚，不再把非法节次状态留在界面里。
- 时间模板现在会正确显示主时间表和副时间表的具体使用情况，便于定位是哪张课表、哪节课在引用。
- 关于页新增超级岛诊断日志开关，支持导出日志与清空后重新收集，便于用户反馈问题。
- 超级岛左侧文字图支持更细粒度的水平、垂直偏移调节。

## 问题修复

- 修复时间模板二级弹窗、时间选择器在某些情况下出现在底部弹窗后方的问题。
- 修复自动保存后，展开态自定义图片未选择时仍可能保存半成品状态的问题。
- 修复时间模板编辑时名称为空或节次数不合法时，界面没有立即回滚的问题。
- 移除“从最近任务隐藏应用”功能，避免导致超级岛不稳定、意外消失或应用冷启动。
- 优化超级岛诊断日志记录与导出链路，减少“未生成日志却提示没有日志”的情况。
